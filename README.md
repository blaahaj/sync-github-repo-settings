# @blaahaj/sync-github-repo-settings

Manage GitHub repository settings as code.

Specifically, what can be managed is:

- description
- archival
- visibility
- topics

## Setup

Requires `node` and `pnpm`. See `.tool-versions` for the versions this has been tested against.
